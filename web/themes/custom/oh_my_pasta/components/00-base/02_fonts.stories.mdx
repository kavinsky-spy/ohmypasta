# Typography
The site is using [Lato](https://fonts.google.com/specimen/Lato) and [Catamaran](https://fonts.google.com/specimen/Catamaran) as the main fonts. To load the font on Storybook, we use the `preview-head.html` file.
In order to load it in Drupal, we are following the Drupal standards:
- The font stylesheet is added as an external library in the `libraries.yml` file of the theme.
- This new library is added as a dependency of the global library, so it loads on every single page.
- We are preconnecting the google domains related in the `html.html.twig` template.
